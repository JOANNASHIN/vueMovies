# nowPlaying
작업기간: 2021.11.01 ~ 2021.11.19
프로그래밍언어: HTML5, SCSS, Vue.js, Firebase

현재 상영중인 tv 프로그램을 확인할 수 있는 페이지입니다.
pc/mo 반응형 웹으로 구현되었으며, 현재 인기 상영중인 리스트와 프로그램의 상세설명 등을 볼 수 있습니다.
필터를 사용하여 다양한 채널의 현재 상영중인 프로그램들을 확인할 수 있습니다.
